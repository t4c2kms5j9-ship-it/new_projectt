# 🏋️ Тренировки

Личный PWA-трекер силовых тренировок. Все данные хранятся локально в браузере.

## Установка на iPhone

1. Открой ссылку GitHub Pages в Safari
2. Нажми кнопку «Поделить» → «На экран «Домой»»
3. Приложение работает оффлайн

## Активация GitHub Pages

Settings → Pages → Source: GitHub Actions
